## Jobnetwork

Job Network website

## Installation

``` 
    git clone https://gitlab.com/iamfaysal/jobnetwork.git 
```
```
    cd jobnetwork
```
```
    composer inatall
```
```
    cp .env.example .env
```
```
    php artisan key:generate
```

**Create database and update the <pre><code>.env</code></pre> file

```
    php artisan serve
``` or create vhost

**Additional if it needs to add CSS/JS: ```npm install```
